Swiggy Data Cleaning \& Visualization Project



\-> Project Overview



This project focuses on cleaning, analyzing, and visualizing a Swiggy restaurant dataset using Python. The objective was to identify delivery trends, pricing patterns, restaurant ratings, and food category insights through exploratory data analysis.



\-> Technologies Used



1\. Python

2\. Pandas

3\. Matplotlib

4\. Seaborn

5\. Google Colab



\-> Project Workflow



1\. Loaded and explored the dataset.

2\. Checked missing values and duplicate records.

3\. Analyzed numerical and categorical features.

4\. Investigated delivery-time outliers.

5\. Visualized ratings, pricing, and city distributions.

6\. Generated business insights from the dataset.



\-> Key Insights



1\. Kolkata had the highest number of restaurants in the dataset.

2\. Most restaurant prices were concentrated between ₹0–₹500.

3\. Delivery times were mostly between 40–65 minutes.

4\. An unusually high number of restaurants had a 2.9 rating, suggesting possible placeholder/default values.

5\. Indian cuisine was the most dominant food category.



\-> Files Included


1. swiggy.csv (Original)
2. cleaned\_swiggy\_data.csv (After cleaning)

\-> Conclusion



This project helped in understanding practical data cleaning, exploratory data analysis, and visualization techniques using Python libraries.
